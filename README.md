# Excel Report Automation System (Python)

This project automatically cleans messy Excel files and generates a professional business report with summaries, top transactions, and formatting.

## What this solves
Most businesses receive Excel files that are:
- Messy
- Full of errors
- Hard to analyze
- Time-consuming to summarize

This tool turns raw Excel files into a clean, formatted report in one click.

## Features
- Automatically cleans dirty Excel data
- Handles missing values and text in numeric fields
- Groups data by category
- Finds top 5 highest transactions
- Generates a formatted Excel report
- Works with different column naming styles (Total, Amount, Cost, etc.)
